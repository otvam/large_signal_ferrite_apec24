EPCOS TDK N87 - Large-Signal Permeability
=========================================

Measured with a R22 core.
Measured at a temperature of 25 C.

Dataset source:
    - MagNet: Data Driven Methods for Magnetic Core Loss Modeling
    - Princeton University, Dartmouth College, and Plexim GmbH, 2023
    - https://mag-net.princeton.edu

Variable description:
    - f_vec - frequency vector in Hz (size n_f)
    - B_mat - peak flux density matrix (size n_f x n_B)
    - mu_mat - complex permeability matrix (size n_f x n_B)
